Prochaines étapes recommandées

1

Pipeline GitHub + Vercel

Priorité absolue — c'est le socle de tout. Une fois le remote lié, chaque commit déclenche un déploiement automatique. Plus de `vercel --prod` à la main.

git remote add origin https://github.com/[user]/parcachien.git
2

SEO dynamique par parc

Ajouter generateMetadata() dans app/parcs/[slug]/page.tsx avec titre, description et Open Graph par ville. Impact direct sur le trafic organique Google.

app/parcs/[slug]/page.tsx
3

Clustering carte

npm install react-leaflet-markercluster. Wrapper les marqueurs dans un MarkerClusterGroup. La carte devient lisible dès le zoom national.

npm install react-leaflet-cluster
4

Récupération Blog / Landing live

Fetch ou copie manuelle des pages existantes sur www.parcachien.com pour ne pas perdre le juice SEO accumulé. Créer /app/blog/ en local.

app/blog/[slug]/page.tsx
